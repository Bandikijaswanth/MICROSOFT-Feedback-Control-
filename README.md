# MICROSOFT-Feedback-Control-
AIM

To compute the control error in a feedback control system.

PROCEDURE
Initialize setpoint and actual values
Apply the error formula: error = setpoint − actual
Substitute the given values
Compute the error
Display the result
CODE
# Input values
setpoint = 10
actual = 7

# Error calculation
error = setpoint - actual

# Output
print("Error:", error)
OUTPUT

Error: 3

RESULT

The control error is 3, which is used by the controller to adjust the system toward the desired setpoint.
